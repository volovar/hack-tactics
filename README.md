# Week Planner
An app to plan your tasks for the week.

## Setup
1. Clone the repo
    ```bash
    git clone https://github.com/volovar/week-planner.git
    ```
1. Move into the directory
    ```bash
    cd week-planner
    ```
1. Install packages
    ```bash
    npm i
    ```
1. Start the app
    ```bash
    npm start
    ```
