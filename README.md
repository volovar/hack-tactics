# Tactics Coder
An app to plan your tasks (coding or otherwise) for the week. Become a master tactician and always know what to do next.

## Setup
1. Clone the repo
    ```bash
    git clone https://github.com/volovar/tactics-coder.git
    ```
1. Move into the directory
    ```bash
    cd tactics-coder
    ```
1. Install packages
    ```bash
    npm i
    ```
1. Start the app
    ```bash
    npm start
    ```
